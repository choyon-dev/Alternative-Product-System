Custom Elementor Widget to Display Alternative Products with Modern Design using Flexbox.

== Description ==

Elementor Alternatives Widget is a custom plugin that adds a new Elementor widget to display alternative products defined by an ACF Relationship field. It supports flexible styling and modern design with responsive layouts.

**Features:**
- Query alternative products using ACF Relationship fields.
- Display results in a list view with modern, Flexbox-based design.
- Fully customizable in Elementor's widget settings.
- Compatible with WooCommerce and standard WordPress post types.
- Optimized for responsive layouts.

**Use Cases:**
- Display alternative products for WooCommerce products.
- Highlight related items for better navigation and sales.
- Enhance product pages with a sleek, list-style layout.

== Installation ==

1. Upload the plugin folder `elementor-alternatives-widget` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Ensure Advanced Custom Fields (ACF) is installed and activated.
4. Create a relationship field in ACF for your post type and populate it with related items.
5. Open Elementor, drag the "Alternative Products" widget, and configure its settings.

== Frequently Asked Questions ==

= What is required to use this widget? =
This plugin requires Elementor (Free or Pro) and ACF Pro for the relationship fields.

= Can I customize the design further? =
Yes, you can. The plugin includes Flexbox-based CSS, which you can override in your theme’s custom CSS if needed.

= Does it work for all post types? =
Yes, it can be used for any post type that supports ACF relationship fields.
